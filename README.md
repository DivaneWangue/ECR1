# SkillHub – Landing Page

## Présentation du projet

Cette landing page a été réalisée dans le cadre d’un projet de développement web,elle a pour objectif de présenter la plateforme **SkillHub**.

La page met en avant les principales fonctionnalités de la plateforme, présente plusieurs formations disponibles et incite les utilisateurs à accéder à l’application.


## Objectifs du projet

Le projet vise à :

* Présenter la plateforme SkillHub
* Expliquer ses fonctionnalités principales
* Afficher une liste de formations
* Proposer une interface moderne et responsive
* Intégrer des interactions JavaScript


## Fonctionnalités de la landing page

### Présentation de la plateforme

Une section **Hero** introduit la plateforme SkillHub avec un titre, une description et un bouton d’appel à l’action.

### Fonctionnalités principales

Une section dédiée présente plusieurs fonctionnalités de la plateforme :

* Catalogue de formations
* Suivi de progression
* Recommandations personnalisées

### Affichage dynamique des formations

Les formations sont chargées dynamiquement depuis un fichier JSON grâce à **JavaScript et une requête fetch()**.

Chaque formation contient :

* un titre
* une catégorie
* une durée
* un niveau

### Call To Action

Une section incite l’utilisateur à découvrir la plateforme ou à consulter les formations disponibles.

### Footer

Le footer contient :

* le nom de la plateforme
* un lien vers les mentions légales
* l’année


## Technologies utilisées

* **HTML5** (structure de la page)
* **CSS3** (mise en page et responsive design)
* **JavaScript** (interactions et chargement dynamique des données)
* **JSON** (stockage des données des formations)

## Accessibilité

La page respecte les bonnes pratiques d’accessibilité :

* utilisation de balises HTML sémantiques (`header`, `nav`, `main`, `section`, `footer`)
* utilisation de l’attribut `alt` pour les images
* hiérarchie logique des titres (`h1`, `h2`, `h3`)
* navigation claire et compréhensible


## Responsive Design

La page est entièrement responsive et s’adapte à différents types d’écrans :

* ordinateur (desktop)
* tablette
* mobile



## Interactions JavaScript

La page intègre plusieurs interactions :

* ouverture / fermeture du menu mobile
* chargement dynamique des formations via `fetch()`
* défilement fluide vers la section des formations


## Lancer le projet

1. Télécharger ou cloner le projet
2. Ouvrir le dossier dans un éditeur de code (ex : VS Code)
3. Lancer un serveur local (Live Server recommandé)
4. Ouvrir le fichier `index.html` dans le navigateur
